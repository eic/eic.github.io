##### Project eAST {#project-east}

{% include navigation/findlink.md name='east' tag='Project eAST' %}
(eA Simulation Toolkit) is led by Makoto Asai (SLAC) and builds on the work done in existing detector simulations for the EIC.
To ease leveraging new and rapidly evolving computing technologies, it implements a common and integrated approach for fast and full detector simulations in Geant4 with a modular plug-and-play design.
Members of the EICUG Software Group are actively involved in this project.
